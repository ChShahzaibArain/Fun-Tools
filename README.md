# Fun-Tools
This tool for fun it's only working on termux 

# Requirements:

* python

* termux

* Non root

# How to install:

* `pkg update`

* `pkg upgrade`

* `pkg install git`

* `pkg install python python2`

* `git clone https://github.com/ChShahzaibArain/Fun-Tools`

# How to use

* `cd Fun-Tools`

* `chmod +x fun.py`

* `python fun.py`
